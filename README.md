# SkySkins 🐾

The definitive 3D web viewer for **Hypixel SkyBlock** pet skins. Preview animations, rarity tiers, and mythic cosmetics with real-time Three.js rendering directly in your browser.

[**Visit SkySkins**](https://skyskins.github.io)

---

## 🌟 Features

- **Real-time 3D Rendering**: High-performance Minecraft-style pet models using Three.js.
- **Animated Skins**: Support for frame-by-frame skin animations.
- **Day/Night Cycles**: Automatic and manual toggles for skins that change appearance based on world time.
- **Full Database**: Easy search and filtering for all SkyBlock pets.
